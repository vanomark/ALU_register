# ALU_register

![Verilog](https://img.shields.io/badge/Verilog-IEEE%201364-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

> **ALU_register** — параметризуемый модуль арифметико-логического устройства (АЛУ) с регистром на Verilog.

## 📋 Описание

Проект реализует 8-битное (с возможностью изменения разрядности) АЛУ с синхронным выходным регистром и асинхронным сбросом. Модуль поддерживает 8 базовых логических и арифметических операций, выбираемых через 3-битный код операции (`opcode`).

## 📁 Структура репозитория

```
├── alu_register.v        # Исходный код модуля АЛУ
├── alu_register_tb.v     # Тестбенч для верификации
├── dump.vcd              # Файл трассировки сигналов (VCD)
└── README.md             # Документация
```

## ⚙️ Интерфейс модуля

### Параметры
| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|----------|
| `WIDTH`  | 8                    | Разрядность операндов и результата |

### Порты
| Порт | Направление | Ширина | Описание |
|------|------------|--------|----------|
| `clk_i`   | input  | 1      | Тактовый сигнал (положительный фронт) |
| `arstn_i` | input  | 1      | Асинхронный сброс (активный низкий) |
| `first_i` | input  | WIDTH  | Первый операнд |
| `second_i`| input  | WIDTH  | Второй операнд |
| `opcode_i`| input  | 3      | Код операции |
| `result_o`| output | WIDTH  | Результат вычисления |

## 🔧 Поддерживаемые операции

| `opcode` | Операция | Описание | Пример |
|----------|----------|----------|--------|
| `3'b000` | NOR      | Побитовое ИЛИ-НЕ | `~(first \| second)` |
| `3'b001` | AND      | Побитовое И | `first & second` |
| `3'b010` | ADD_S    | Сложение со знаком | `$signed(first) + $signed(second)` |
| `3'b011` | ADD_U    | Сложение без знака | `first + second` |
| `3'b100` | NOT      | Побитовое НЕ второго операнда | `~second` |
| `3'b101` | XNOR     | Побитовое ИЛИ-ИСКЛючающее-НЕ | `~(first ^ second)` |
| `3'b110` | EQ       | Сравнение на равенство | `(second == first) ? 1 : 0` |
| `3'b111` | SHR      | Логический сдвиг вправо | `first >> second` |

> ⚠️ **Примечание**: Результат операции сравнения (`EQ`) записывается в младший бит выхода, остальные биты заполняются нулями.

## 🧪 Тестирование

Проект включает тестбенч `alu_register_tb.v`, который демонстрирует:
- Инициализацию сигналов
- Подачу тестовых векторов
- Генерацию тактового сигнала
- Сброс модуля
- Запись трассировки в файл `dump.vcd`

### Запуск симуляции (Icarus Verilog)

```bash
# Компиляция
iverilog alu_register.v alu_register_tb.v

# Просмотр результатов (например, в GTKWave)
gtkwave dump.vcd

# Запуск симуляции
vvp alu_sim
```
